# ADR 001: Выбор системы аутентификации

## Контекст
Нужно выбрать способ аутентификации пользователей.

## Решение
Используем JWT токены с refresh tokens.

## Обоснование
1. Stateless - легко масштабировать
2. Поддержка мобильных клиентов
3. Стандартный подход для REST API

## Последствия
- Нужно хранить refresh токены в БД
- Токены должны быть короткоживущими
- Нужен механизм отзыва токенов
